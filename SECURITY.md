# Security Policy                                                            
                                                                               
  ## Reporting a Vulnerability                                                 
                                                                               
  If you discover a security vulnerability in FlowForge-Track, please report it 
  by emailing e.covasa@me.com.                 
                                                                               
  **Please do not** report security vulnerabilities through public GitHub      
  issues.                                                                      
                                                                               
  ## Supported Versions                                                        
                                                                               
  | Version | Supported          |                                             
  | ------- | ------------------ |                                             
  | 0.1.x   | :white_check_mark: |                                             
                                                                               
  ## Security Updates                                                          
                                                                               
  We take security seriously. Security patches will be released as soon as     
  possible after a vulnerability is confirmed
